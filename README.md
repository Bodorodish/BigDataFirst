Шаг 1: Поднять базу
В папке с проектом выполнить в терминале:
```
docker-compose up -d
```

Шаг 2: Залить исходник
В DBeaver нажми ПКМ на базу -> Import Data -> CSV из исходных данных.

Шаг 3: Создать структуру (DDL)
Скрипт создания таблиц: выполнить ddl.sql

Шаг 4: Разнести данные (DML)
Скрипт для создания фактов: выполнить dml.sql

Шаг 5: Проверить
Выполнить final.sql, должно быть ±10000 записей
